## ~10:30–11:25 AM — 3D Garment Viewer Session (continued)

### Major work done this session:

**Real GLB model integration:**
- David found a proper Sketchfab hoodie GLB. Inspected structure: 4 meshes, single material `FABRIC_3_FRONT_1850`, roughness 0.98, metalness 0, 3 UV sets
- Copied to `public/models/hoodie.glb`
- Rewrote `HoodieViewer3D.tsx` to use `useGLTF` (react-three/drei) loading real model instead of extruded SVG

**Seamless UV fix (key technical win):**
- Problem: Each mesh part (body, sleeves, hood) has separate UV islands → pattern breaks at seams
- Solution: World-space UV projection — compute bounding box of ALL mesh parts in world space, remap every vertex UV to normalised world X/Y position
- Since adjacent vertices at seam edges share world position → identical UVs → completely seamless tiling
- This is the same approach needed for Printful cutting templates later

**Refactored to generic `GarmentViewer3D`:**
- `components/merch/GarmentViewer3D.tsx` — accepts `modelPath` prop, works for any garment GLB
- `HoodieViewer3D.tsx` now just re-exports from GarmentViewer3D (backward compat)

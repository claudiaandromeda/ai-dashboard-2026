### Lessons Learned (for future setups)
1. **Never paste tokens in chat** — use `openclaw config set` directly
2. **Always choose the simplest route** — David's new rule
3. **Discord bot ≠ Discord user account** — they're separate things
4. **Error 4014** = Privileged Gateway Intents not enabled
5. **"Failed to resolve Discord application id"** = bad/expired token
6. **"no-mention" skip** = bot requires @mention in guild channels by default
7. **guildId is not a valid config key** — use `guilds` object instead
8. **Don't guess CLI commands** — read the schema/help first
